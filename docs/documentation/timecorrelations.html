Two-time correlation functions¶

function correlation(tspan, rho0, H, J, op1, op2; Gamma, Jdagger, tmp)¶

Calculate two time correlation values \(\langle A(t) B(0) \rangle\)

The calculation is done by multiplying the initial density operator with \(B\) performing a time evolution according to a master equation and then calculating the expectation value \(\mathrm{Tr} \{ A \rho \}\)

Parameters:
  • tspan – Points of time at which the correlation should be calculated.
  • rho0 – Initial density operator.
  • H – Operator specifying the Hamiltonian.
  • J – Vector of jump operators.
  • op1 – Operator at time t.
  • op2 – Operator at time t=0.
Keyword Gamma:

Vector or matrix specifying the coefficients for the jump operators.
Keyword Jdagger (optional):
 

Vector containing the hermitian conjugates of the jump operators. If they are not given they are calculated automatically.
Keyword kwargs:

Further arguments are passed on to the ode solver.
function correlation(rho0, H, J, op1, op2; eps, h0, Gamma, Jdagger, tmp)¶

Calculate two time correlation values \(\langle A(t) B(0) \rangle\)

The calculation is done by multiplying the initial density operator with \(B\) performing a time evolution according to a master equation and then calculating the expectation value \(\mathrm{Tr} \{ A \rho \}\). The points of time are chosen automatically from the ode solver and the final time is determined by the steady state termination criterion specified in steadystate.master.

Parameters:
  • rho0 – Initial density operator.
  • H – Operator specifying the Hamiltonian.
  • J – Vector of jump operators.
  • op1 – Operator at time t.
  • op2 – Operator at time t=0.
Keyword eps:

Tracedistance used as termination criterion.
Keyword h0:

Initial time step used in the time evolution.
Keyword Gamma:

Vector or matrix specifying the coefficients for the jump operators.
Keyword Jdagger (optional):
 

Vector containing the hermitian conjugates of the jump operators. If they are not given they are calculated automatically.
Keyword kwargs:

Further arguments are passed on to the ode solver.
function spectrum(omega_samplepoints, H, J, op; rho0, eps, rho_ss)¶

Calculate spectrum as Fourier transform of a correlation function

This is done by the use of the Wiener-Khinchin theorem

\[S(\omega, t) = \int_{-\infty}^{\infty} d\tau e^{-i\omega\tau}\langle A^\dagger(t+\tau) A(t)\rangle = 2\Re\left\{\int_0^{\infty} d\tau e^{-i\omega\tau}\langle A^\dagger(t+\tau) A(t)\rangle\right\}\]

The argument omega_samplepoints gives the list of frequencies where \(S(\omega)\) is caclulated. A corresponding list of times is calculated internally by means of a inverse discrete frequency fourier transform. If not given, the steady-state is computed before calculating the auto-correlation function.

Parameters:
  • omega_samplepoints – List of frequency points at which the spectrum is calculated.
  • H – Operator specifying the Hamiltonian.
  • J – Vector of jump operators.
  • op – Operator for which the auto-correlation function is calculated.
Keyword rho0:

Initial density operator.
Keyword eps:

Tracedistance used as termination criterion.
Keyword h0:

Initial time step used in the time evolution.
Keyword Gamma:

Vector or matrix specifying the coefficients for the jump operators.
Keyword Jdagger (optional):
 

Vector containing the hermitian conjugates of the jump operators. If they are not given they are calculated automatically.
Keyword kwargs:

Further arguments are passed on to the ode solver.

function correlation2spectrum{T <: Number}(tspan, corr; normalize)¶

Calculate spectrum as Fourier transform of a correlation function with a given correlation function

Parameters:
  • tspan – Time list corresponding to the correlation function.
  • corr – Two-time correlation function.
Keyword normalize (optional):
 

Specify whether or not to normalize the resulting spectrum to its maximum; default is false.

As a brief example, say we want to calculate the two-time correlation function of a cavity field, i.e. \(g(t) = \langle a(t) a^\dagger(0)\). We can do this with:

basis = FockBasis(10)
a = destroy(basis)
κ = 1.0
η = 0.3
H = η*(a + dagger(a))
J = [sqrt(2κ)*a]

tspan = [0:0.1:10;]
ρ0 = fockstate(basis, 0) ⊗ dagger(fockstate(basis, 0))
g = timecorrelations.correlation(tspan, ρ0, H, J, dagger(a), a)

If we omit the list of times tspan, the function automatically calculates the correlation until steady-state is reached:

t_s, g_s = timecorrelations.correlation(ρ0, H, J, dagger(a), a)

To calculate the spectrum, we can either use the timecorrelations.correlation2spectrum to calculate it from a given correlation function, or alternatively, one can use timecorrelations.spectrum, which calculates the correlation function internally and returns the Fourier transform.
